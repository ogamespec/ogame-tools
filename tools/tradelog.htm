12 Октября 2010v0.27 - Актуальность заявок. По умолчанию заявка самоуничтожается через 7 дней.
- В настройках можно указать свой альянс.
- Новая система смены пароля в 2 этапа. В первом письме пользователь получает ссылку для смены. При открытии ссылки пользователь получает второе письмо с новым паролем.
4 Октября 2010v0.26 - Сортировка предложений на стартовой странице
- Исправлена ошибка с регистрацией нового пользователя
3 Октября 2010v0.25 - Улучшена работа с языками, странами и скинами: страна выбирается на стартовой странице. Она задаёт список доступных вселенных для выбора. Также на стартовой странице можно выбрать глобальный язык и скин. Глобальные настройки скина и языка можно заменить в настройках пользователя.
- Подсвечивается текущая выбранная страна
- Добавлено несколько новых языковых ключей
3 Октября 2010v0.24 - На стартовой странице показываются последние предложения
- Исправления в базе данных
- Обновлены курсы вселенных (bg, en) и добавлены некоторые страны
- При отсутствии перевода по умолчанию используется английский
- Поменял название утилиты на Торговый Терминал
- Исправлено дублирование заявок после редактирования
- Турецкий машинный перевод
- Украинский перевод
2 Октября 2010v0.23 - Добавлены вселенные из редизайна + возможность добавлять новые
- Возможность изменять курсы индивидуально для каждой вселенной
- Упростился выбор языка и списка вселенных для выбранной страны
- Некоторые неработающие скины заменены на локальные версии
- Из списка транспортов убран Колонизатор
20 Июля 2009v0.22 - Система восстановления забытых паролей
- Машинный перевод для: Китайский, Греческий, Немецкий
- Добавление заявки выполняется AJAX-запросом, поэтому при нажатии на F5 не происходит повторной отправки
- Исправлена ошибка в администрировании пользователей
11 Мая 2009v0.21 - Добавление серверов других стран
- Возможность изменить скин в настройках (с проверкой правильности URL на эксплоиты)
- Подсказка в главном меню для переводчиков
- Возможность продавать и покупать лунопопытки
- Добавлено несколько скинов (список будет постоянно пополняться)
- Гостевой вход (любое имя пользователя без пароля)
- Изменено время редиректа после выхода с 3 на 2 секунды (может вообще убрать?)
- Перевод базы данных на UTF-8 (до этого UTF-8 был только на HTML-страницах)
- Добавление даты заявки
- Подчищен HTML-код формы подачи заявки, теперь нет конфликтов со скинами
- Проверка параметров при добавлении заявки на PHP и SQL-инъекции
- Теперь курс можно писать через двоеточие (напр. 2:1 или 1:1.3). Можно без.
- Убрал ненужный CSS (default.css), исправил задний фон для оверлеев.
- Длительность сессии теперь 24 часа.
- Курс можно писать через запятую, напр. 1:2,1
- Автоматический калькулятор обмена
2 Мая 2009v0.20 - Начато отслеживание истории изменений
- При наведении мышки на ссылку пользовательского меню появляется подсказка
- Исправлено отображение DIV-ов на главной странице для IE
- Администраторы могут добавлять заявки. Супер-администраторы не могут.
- Добавлен запрет кеширования страницы (лечит баги с сессиями в старых IE)
- Меню пользователя (изменить имя, изменить язык)
- Статистика по количеству пользователей и заявок на главной странице
- Улучшение файла локализации (перевод всех HTML-entities на utf-8)
- Расширение списка языков
