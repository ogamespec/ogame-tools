Скрипт предназначен для русификации игрового интерфейса немецкой вселенной 20.DE. Для установки скрипта в Опере 10.50+ проделайте следующие действия:

1. Настраиваем оперу для выполнения пользовательских скриптов:
Заходим в настройки: щелчок левой кнопкой на красной блямбе с буквой О -> Настройки -> Настройки или нажать сочетание Ctrl+F12
Далее: Содержимое -> Настроить JavaScript -> Папка пользовательских файлов JavaScript и выбираем заранее созданную папку на диске, где будут храниться скрипты для Ogame (например c:\OperaScripts).


2. Скачиваем в папку со скриптами следующие скрипты:
(Необходимо нажать правой кнопкой по ссылке, выбрать "Сохранить по ссылке как..." и указать папку со скриптами)
Скипт перевода: http://ogamespec.com/tools/Russian20DE.user.js
Вспомогательный скрипт: http://ogamespec.com/tools/jquery.user.js

Готово!
